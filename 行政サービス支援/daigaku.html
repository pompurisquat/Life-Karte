1年生1
必修科目：共済制度論

あなたの「見えない保障」を知る

公務員には民間会社員にはない、強力な医療費補填制度があります。まずここを理解することがすべての出発点です。

🏥
LESSON 1-1
付加給付制度とは何か？
✓
25,000円
教職員・多くの国家公務員は
月の医療費自己負担がこの金額を超えた分が戻ってくる

健康保険には「高額療養費制度」がありますが、公務員の共済組合にはそれよりさらに手厚い「付加給付」があります。月の医療費自己負担が一定額（多くの共済で25,000円）を超えた分を、組合が自動的に補填してくれます。

📌 ポイント：民間の会社員（協会けんぽ）の自己負担上限は月57,600円〜87,430円。公務員はその約2〜3倍手厚いのです。
👨‍👩‍👧
LESSON 1-2
家族の医療費も対象になる
✓
家族全員
被扶養者（配偶者・子ども）の
医療費にも付加給付が適用される

本人だけでなく、共済組合の被扶養者（配偶者・子ども・両親など）の医療費にも付加給付が適用されます。子どもが入院した場合も、自己負担が月25,000円程度に抑えられます。

📌 ポイント：「子どもが病気になったときのために医療保険に入っている」という人は、共済の家族補填を確認してから判断を。重複している可能性が高い。
📋
LESSON 1-3
自分の共済組合を確認する方法
✓
3分
職場の給与明細か
共済組合サイトで今すぐ確認できる

給与明細の「共済組合費」欄を見ると、加入している共済組合がわかります。教員なら各都道府県教職員共済、国家公務員なら所属省庁の共済組合。各組合のサイトで「一部負担金払戻金」「付加給付」と検索すれば自己負担上限額を確認できます。

📌 確認事項：①自己負担の月額上限 ②家族（被扶養者）への適用有無 ③申請が必要か自動か の3点をチェック。
🎉

2年生2
必修科目：保険見直し学

払いすぎの保険料を今すぐ取り戻す

共済がカバーする範囲を理解したら、次は重複する民間保険を整理します。削れた保険料が、あなたの資産形成の原資になります。

🏥
LESSON 2-1
医療保険・がん保険は必要か？
✓
月3,000円
不要な医療保険を解約すると
年間36,000円〜72,000円が浮く

付加給付で月25,000円に抑えられる公務員にとって、民間の入院保険・がん保険の役割は限定的です。特に「入院1日5,000円」タイプの保険は、共済で補われる範囲と重複していることがほとんどです。

📌 チェックリスト：①共済の付加給付上限額を超えた分をカバーしているか ②差額ベッド代・先進医療など共済外の費用に特化しているか。「NO」なら見直しを検討。
💔
LESSON 2-2
死亡保険の正しい考え方
✓
遺族年金
公務員が亡くなった場合
遺族に手厚い年金と退職金が支給される

公務員には遺族共済年金・退職手当があります。子どもがいる場合は遺族年金がさらに手厚くなります。「万が一のための死亡保険」は、まず遺族年金と退職金でいくらカバーされるかを試算してから、不足分だけを民間保険で補う設計が正解です。

📌 公式試算ツール：各共済組合のサイトで「遺族年金シミュレーター」が利用可能。まず試算してから保険を考える順番で。
🚨
LESSON 2-3
就業不能保険は職種で判断する
✓
職種で異なる
警察・消防・自衛官は
怪我・事故リスクが一般行政職より高い

デスクワーク中心の行政職は就業不能リスクが低く、共済の傷病手当金（最大3年）でカバーできるケースが多いです。一方、警察官・消防士・自衛官は職業上の危険があるため、就業不能保険の検討価値があります。

📌 判断基準：共済の傷病手当金（給料の2/3程度）が何年間支給されるかを確認。それ以降の長期就業不能に備えるかどうかで判断する。
🎉

3年生3
必修科目：資産形成論

iDeCo・NISAで税金を味方につける

保険料が整理できたら、余剰資金を増やす仕組みを作ります。公務員が使える非課税制度を最大限に活かしましょう。

🏦
LESSON 3-1
公務員のiDeCo活用術
✓
月12,000円
公務員のiDeCo上限額。
年間最大27,600円の節税効果（年収600万の場合）

iDeCoは掛金が全額所得控除になる、公務員が最も活用すべき制度です。月12,000円を20年積み立てると、節税だけで50万円以上の差が生まれることも。60歳まで引き出せない点と、退職金との受け取り方（一時金・年金）の税制を事前に確認しておきましょう。

📌 注意点：2024年12月から企業型DC加入者と同様のルール変更が予定されています。最新情報は加入している共済組合か金融機関で確認を。
📈
LESSON 3-2
新NISAで非課税を最大化する
✓
1,800万円
新NISAの生涯非課税枠。
運用益・配当に税金ゼロで受け取れる

2024年から始まった新NISAは「つみたて投資枠（年120万円）」と「成長投資枠（年240万円）」の2つの枠があります。公務員は毎月の安定収入があるため、積み立て型の長期運用と相性が抜群です。

📌 優先順位：①iDeCo（節税効果が高い）→ ②NISA（柔軟に引き出せる）の順で埋めるのが基本戦略。両方を上限まで使える家庭は最強。
🌍
LESSON 3-3
投資の3原則：長期・分散・低コスト
✓
年0.1%未満
インデックスファンドの信託報酬目安。
コスト差が30年で数百万円の差になる

投資で失敗する多くの原因は「短期売買」「集中投資」「コストの高い商品」です。全世界株式や全米株式に連動するインデックスファンドをNISA・iDeCoで長期積み立てするだけで、プロの大半に勝てることが統計的に示されています。

📌 具体的な選び方：「eMAXIS Slim」「SBI・V」シリーズなど信託報酬0.1%未満のインデックスファンドを選ぶ。個別株・FXは余剰資金の一部のみ。
🎉

4年生4
必修科目：老後設計学

退職金・年金から逆算する人生設計

ゴールを知ることで、今何をすべきかが見えます。公務員の退職金・共済年金の仕組みを理解して、老後の「不足分だけ」を自分で準備しましょう。

💰
LESSON 4-1
公務員の退職金はいくら？
✓
約2,000万円
国家公務員が定年まで勤めた場合の
平均退職金の目安（60歳定年・等級による）

公務員の退職金は「退職手当基本額（俸給月額 × 支給率）」で計算されます。支給率は勤続年数と退職理由（自己都合・定年）によって大きく変わります。20年以上勤めると定年退職の支給率が大幅に上がるため、途中退職との差が数百万円になることも。

📌 試算方法：人事院のWebサイト「退職手当額の計算」で現在の等級と勤続年数を入力すれば概算が出ます。退職後のライフプランはここから逆算を。
🧓
LESSON 4-2
共済年金から厚生年金への移行を理解する
✓
職域加算
2015年廃止→「年金払い退職給付」に移行。
公務員の年金は3階建て構造

2015年10月から公務員の年金制度が大きく変わりました。「①国民年金（基礎）」「②厚生年金（民間と統一）」「③年金払い退職給付（公務員独自）」の3階建てです。かつての共済年金の「職域加算」は廃止されましたが、代わりの上乗せ制度が存在します。

📌 確認先：ねんきん定期便（毎年誕生月に届く）で予想受給額を確認。50歳以上は将来の見込み額が記載されています。
🗺️
LESSON 4-3
老後の「不足分だけ」を準備する
✓
逆算思考
老後に必要な総額から
年金・退職金を引いた「不足分」が自分で準備する額

「老後2,000万円問題」はあくまで一般的な試算です。公務員は退職金と年金が手厚いため、実際の不足額は民間より少ないことがほとんどです。自分の退職金見込み額と年金見込み額を合算して、老後の生活費（月25〜30万円 × 30年）と比較することで、本当に必要な積立額が決まります。

📌 計算式：（老後月額支出 × 12か月 × 30年）−（退職金 + 年金受給総額）= 自分で準備すべき金額。iDeCo・NISAでこの額を目指す。
